# 智能净水节水花盆功能（挑战杯参赛项目）

项目闭源，如需提供请与作者联系。

1.土壤缺水时自动浇水。

2.天黑时自动开启红蓝光为植物提供光合作用。

3.实时获取环境温度，湿度与土壤湿度并显示。

4.连接WIFI实时上传获取的数据到云端。

5.用户可以通过网页获取植物生长数据。

6.用户可以通过小程序获取实时数据。

7.将植物生长数据转换成表情显示。

8.支持太阳能充电减少电能消耗。

9.支持锂电池供电。

10.通过活性炭，沸石的物理性对水进行过滤。

11.回收土壤过饱和水资源，循环利用。
